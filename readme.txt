замечание: возможны проблемы при обновлении firmware на модеме.
при перепрошивке с B01 -> B05 застреваем на 19% из-за того, что устройство после рестарта определяется некорректно.
Нужно, не закрывая обновлялку, установить драйвер (из комплекта драйверов для модема) - тип ZTE Diagnostic port.
После этого обновление пройдет до конца успешно.

http://192.168.0.1/goform/goform_process?goformId=MODE_SWITCH&switchCmd=FACTORY

2. Для перевода модема в исходный режим отправить команду (например putty) на ZTE NMEA Device:
AT+ZCDRUN=F
